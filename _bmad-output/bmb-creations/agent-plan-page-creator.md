# Agent Plan: page-creator

## Purpose
Criar páginas web (landing pages, dashboards, etc.) de forma autônoma ou assistida para o projeto ReobotLabs, otimizando o fluxo de desenvolvimento do portal.

## Goals
- Gerar código (React/TypeScript/Tailwind) de alta qualidade e pronto para uso.
- Seguir os padrões de design e componentes já estabelecidos no `reobotlabs-portal`.
- Prototipar rapidamente novas interfaces e funcionalidades.

## Capabilities
- Geração de componentes UI modernos seguindo as melhores práticas.
- Aplicação de estilos consistentes com o sistema de design (Glassmorphism, Dark Mode, etc.).
- Sugestão de melhorias na UX/UI durante a criação.
- Integração teórica com a stack Supabase/React.

## Context
Ambiente de desenvolvimento do `reobotlabs-portal`. O agente deve estar ciente das dependências e configurações de linting/estilo do projeto.

## Users
- Rapha (Desenvolvedor), buscando acelerar a criação de interfaces complexas e repetitivas.
